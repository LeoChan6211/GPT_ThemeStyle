# GPT Style Theme (Extension)

純 HTML 無法把 `chatgpt.com` 內嵌後再改 CSS（網站會用防嵌入政策阻擋）。這個 Chrome/Edge 擴充套件會把你的灰階低對比 CSS **長期注入**到 ChatGPT 頁面，重新整理也不會消失。

## 安裝（Chrome / Edge）

1. 打開 `chrome://extensions`（Edge 是 `edge://extensions`）
2. 開啟「開發人員模式」
3. 點「載入未封裝項目」
4. 選擇資料夾：`gpt-gray-theme-extension`
5. 點右上角擴充套件圖示 →「GPT Gray Theme」→「開啟設定」調整顏色/亮度/對比（或到擴充套件的「詳細資料」→「擴充套件選項」）

> 小提醒：`options.html` 不能用「直接雙擊檔案」的方式打開，必須從擴充套件的「選項」進入，否則會因為缺少 `chrome.storage` 而看起來像白屏。

## 輸入框去哪了？

本擴充套件會把輸入列做成「右下角彈出」：

- 平常只會看到右下角一個小對話泡泡按鈕
- 滑鼠移到按鈕上就會彈出完整輸入列
- 滑鼠移開後若沒有任何操作，閒置約 2 秒會自動收回
- 只要你在輸入列區域內移動滑鼠、點按按鈕、打字，輸入列會保持展開

## 作用範圍

- `https://chatgpt.com/*`
- `https://chat.openai.com/*`

## 右上角「新對話模式」兩顆按鈕

本擴充套件不再把「群組對話 / 暫存對話」硬插進左側側欄（容易被 ChatGPT 的動態重繪打回來、造成重疊）。
目前改成保留原本右上角原生按鈕樣式，並做顯示規則：

- 只在「新對話起始畫面」顯示
- 你點選任一模式後會自動隱藏

## 設定會存在哪裡？

設定會存到 Chrome/Edge 的同步儲存（`chrome.storage.sync`），所以：

- 重新整理不會消失
- 你如果登入同一個瀏覽器帳號，可能會同步到其他電腦（可自行關閉瀏覽器同步）
